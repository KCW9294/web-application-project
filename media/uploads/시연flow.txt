0. 메인페이지
    - 메인 화면
    - 버튼 누르면 서비스 소개 나옴
    - 버튼 누르면 서비스 이용 방법 나옴
    - 버튼 누르면 어바웃 어스 나옴
    - 처음으로 버튼 : 맨 위로 올라갈 수 있음
    - 바로 grow 코칭 실습으로 가려고 하면 할 수 없음. 회원가입 페이지가 뜬다. 
      -> grow 코칭 실습, 대화내역보기, 커뮤니티-설문은 회원가입을 해야 보여짐. 
1. 회원가입하기
    - 아이디 작성 -> 중복확인 가능 
    - 비밀번호 : 비밀번호 작성 조건 문구 출력
    - 비밀번호 : 재작성 시 일치 및 불일치 문구 출력
    - 이메일 : 중복확인 가능
    - 이름 작성
    - 닉네임 : 중복확인 가능
    - 나이 작성 
    - 직무 선택
    - 직급 선택
    - 성별 선택
    - 개인정보 처리방침 확인 및 동의 
    - 만약 필수란 작성하지 않으면 경고 문구가 뜸
2. 로그인하기
    - 아이디 저장 기능
    - 캡챠 기능: 비밀번호를 연속으로 5번 이상 틀렸을 시 보안 문구를 입력할 수 있도록 한다
    - 아이디 찾기 : 아이디를 잊어버렸다면 이름과 메일을 통하여 회원가입 때 작성한 메일로 아이디 안내 메일 발송 
    - 비밀번호 찾기 : 비밀번호를 잊어버렸다면 아이디, 이름, 닉네임을 통하여 회원가입 때 작성한 메일로 비밀번호 재설정 링크가 첨부된 메일 발송
4. grow 연습하기
    - 로그인 안해도 가능 (인데 왜 자꾸 에러가 뜨는 것인가)
5. grow 실습하기
    - 대화 주제 선택 : 만약 선택지가 아닌 다른 주제로 대화를 하고 싶다면 직접 입력 가능
    - 페르소나 설정
    [대화 창] 
    - 어떤 페르소나를 선택했는지 알 수 있음 
    - GROW 예시 질문 가이드
    - 에러 발생할 시 조치 가이드
    - 음성으로 대화 가능 : 사용 방법 팝업 안내 
    - 실시간 점수: 대화를 할 때마다 실시간으로 바뀌어 확인 가능 
    - GROW 순서대로 질문을 하고있는가를 나타내줌 
6. 결과 페이지(intro)
    - 사용자의 대화를 grow모델 기반 결과 및 5가지 코칭 요인으로 분석할 것이라는 소개 및 점수와 간략한 그래프
7. 결과 페이지(main)
    - 나의 점수 소개 및 산출 방법 안내
    - grow 모델 기반 대화 분석 그래프 제공
    - 5가지 코칭 요인 기반 대화 분석 그래프 제공
    - 5가지 코칭 요인 기반 어느 정도로 대화를 했는지 제공 및 조언(?)
    - 설문 요청하기(설문요청방법) : 나의 대화가 grow 기반 설문을 받을 수 있다는 안내 및 설문 받기 방법 소개
8. 대화 내역 보기
    - 앞선 대화 내역 및 결과 확인 가능
    - 설문 요청 기능 : '설문요청하기' 버튼을 누르면 제목과 내용을 작성란이 있고 모두 작성해야 공유가 가능함. 만약 설문을 그만 받고 싶다면 공유 중지 버튼을 클릭하여 중지 가능.  
    - 설문 결과 확인하기 : GROW 모델 기반 설문지를 통한 설문 결과 그래프 제공 및 의견 보기 가능
9. 커뮤니티
    - 공지사항 : 관리자만 작성 가능함. 파일 업로드(관리자) , 다운로드(사용자) 가능
    - 설문 요청 : 다른 사람이 설문 요청한 것 확인 가능. 대화를 보며 GROW 기반 설문지를 통해 설문 가능. 14개의 객관식 및 1개의 주관식. 객관식은 필수, 주관식은 선택. 설문완료하면 완료한 설문이라고 나타남.
10. 관리자 페이지
    - 홈 화면 오른쪽 위에 관리자 계정으로 로그인 했을 시에만 보이는 관리자 페이지 있음. 사용자 통계 및 페르소나 설정 통계 확인 가능.
